# LaTeX 期刊/会议格式切换模板

一个可以快速切换不同期刊/会议格式的 LaTeX 模板，无需重复复制粘贴论文内容。

## 📁 项目结构
├── main.tex # 主文件（此处为通用 article 示例）
├── preamble.tex # 导言区
├── body/ # 论文正文（换模板时不用修改！）
│ ├── abstract.tex
│ ├── introduction.tex
│ ├── Relatedwork.tex
│ ├── method.tex
│ ├── experiment.tex
│ └── conclusion.tex
├── figures/ # 图片文件夹
└── reference.bib # 参考文献
